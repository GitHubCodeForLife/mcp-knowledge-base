# Docs

This folder contains user-facing documentation for the MCP server and the **LLM Wiki** workflow introduced by the `llm-wiki` change.

## Archive reference

Completed OpenSpec changes are archived under `openspec/changes/archive/`. Use archived changes as examples for:
- final spec wording
- how tasks were applied
- what “done” looks like for a change

For example: `openspec/changes/archive/2026-04-19-implement-local-doc-ai-mcp/`.
